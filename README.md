# yoru-store

Графический магазин приложений поверх Pacman и Flatpak.

## Будет содержать (Phase 5)

- Категории приложений
- Поиск
- Установка / удаление одной кнопкой
- Информация о безопасности (размер, origin, разрешения)

## Стек

Планируется на Kirigami/QML. Альтернативы: нативный GTK (Linguabl), Flutter.

## Принципы

- Никаких «рекомендаций» без явного пинка от пользователя
- Прозрачность: всегда видно, что ставится и откуда
- Без рекламы и проприетарных пейволлов
